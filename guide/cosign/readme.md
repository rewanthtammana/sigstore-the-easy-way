# Cosign

[Cosign](https://github.com/sigstore/cosign) is a signing tool. Its responsibility is to sign the artifacts & if required, push them to the OCI registries. If configured, the signature of the artifact is pushed to the [transparency log](https://github.com/sigstore/rekor).
